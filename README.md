# 🎲 多人联网21点网页游戏

## ✨ 功能特性

✅ **纯网页实现**：不需要下载APP，不需要微信审核，手机电脑直接打开就能玩
✅ **真正多人联网**：支持最多6人同时在线对战，实时同步
✅ **自由坐庄**：开局任何人都可以申请坐庄，每局结束自动轮流坐庄
✅ **4副牌规则**：标准赌场4副牌配置，防止算牌，公平公正
✅ **完整游戏流程**：
  - 登录 → 创建/加入房间 → 准备 → 申请坐庄 → 开始游戏
  - 下注 → 要牌/停牌 → 庄家自动要牌 → 系统自动结算
✅ **积分系统**：初始每人10000积分，最低下注100，只能整百下注
✅ **精美UI**：真实纸牌效果，响应式设计，完美适配手机和电脑
✅ **自带后端**：Node.js + WebSocket 实时通信，部署简单

## 🎮 游戏说明

### 房间流程
1. **创建房间**：输入昵称进入大厅，点击"创建房间"，生成6位房间号
2. **加入房间**：其他玩家输入6位房间号即可加入同一房间
3. **申请坐庄**：玩家可以主动申请成为本局庄家
4. **准备游戏**：所有玩家点击"准备"后，庄家可以开始游戏
5. **轮流坐庄**：每局结束后，庄家自动轮换给下一位玩家

### 游戏流程
1. **下注阶段**：每位玩家选择下注金额（100/200/500/1000/2000/5000）
2. **发牌阶段**：系统给每位玩家发2张牌，庄家第一张牌隐藏
3. **玩家回合**：玩家依次选择"要牌"或"停牌"，超过21点直接爆牌输
4. **庄家回合**：庄家翻开所有牌，小于17点必须要牌，大于等于17点停牌
5. **结算阶段**：系统自动比较点数，判定输赢，实时更新积分
   - 玩家爆牌 → 直接输
   - 庄家爆牌 → 所有没爆牌的玩家都赢
   - 都没爆牌 → 点数大的赢，点数相同平局

## 🌐 公网部署

### 部署到云服务器
1. 把代码上传到你的云服务器
2. 安装依赖：`npm install`
3. 启动服务：`npm start`
4. 开放3000端口（或其他你想要的端口）
5. 访问 `http://你的服务器公网IP:3000` 即可

### 域名配置
可以配置Nginx反向代理，绑定域名，支持HTTPS访问：
```nginx
server {
    listen 80;
    server_name your-domain.com;
    
    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_set_header Host $host;
    }
}
```

## 🎨 自定义修改

### 修改下注金额
编辑 `index.html` 中的下注按钮：
```html
<button class="bet-btn" data-value="100">100</button>
```

### 修改游戏规则
编辑 `game.js` 中的逻辑：
- `calculateHandValue` 方法：修改点数计算规则
- `dealerTurn` 方法：修改庄家要牌规则（默认小于17点必须要牌）

### 修改UI样式
编辑 `style.css` 调整界面颜色、卡片大小、字体等。
